## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动
- **秋季（9-11月）**：天高气爽，是旅游旺季

### 推荐行程

- **一日游**：进贤门 → 揭阳楼 → 揭阳博物馆
- **两日游**：Day1: 进贤门 → 揭阳楼 → 揭阳博物馆；Day2: 黄满寨瀑布

## 交通指南

- **飞机**：揭阳潮汕机场，距市区约20公里
- **高铁**：揭阳站、普宁站
- **自驾**：沈海高速、汕昆高速等可达
