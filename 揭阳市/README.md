# 揭阳市旅游景点指南

## 景点总览

揭阳市位于广东省东部，是粤东地区的重要城市，拥有丰富的自然景观和深厚的历史文化底蕴。揭阳以黄满寨瀑布的壮美、古城古村的历史韵味和独特的潮汕文化而闻名。

## 旅游建议

### 最佳旅游时间
- **春季（3-5月）**：气候宜人，适合户外活动
- **秋季（9-11月）**：天高气爽，是旅游旺季
- **夏季（6-8月）**：适合前往黄满寨瀑布避暑，但需注意台风

### 推荐行程
- **一日游**：进贤门 → 揭阳楼 → 揭阳博物馆
- **两日游**：Day1: 进贤门 → 揭阳楼 → 揭阳博物馆；Day2: 黄满寨瀑布
- **三日游**：可增加惠来靖海古城等景点

### 美食推荐
- 揭阳乒乓粿
- 普宁豆酱
- 惠来绿豆饼
- 揭阳肠粉
- 潮汕工夫茶

## 交通指南

### 到达揭阳
- **飞机**：揭阳潮汕机场，距市区约20公里
- **高铁**：揭阳站、普宁站
- **自驾**：沈海高速、汕昆高速等可达

### 市内交通
- **公交**：覆盖主要景点
- **出租车/网约车**：方便快捷
- **自驾**：景点间自驾方便

## 注意事项

1. 部分景点需提前预约，如揭阳博物馆
2. 前往黄满寨瀑布请注意安全，遵守景区规定
3. 尊重当地风俗习惯，特别是宗教场所
4. 保护环境，不乱扔垃圾
5. 注意防晒和补水，特别是夏季
