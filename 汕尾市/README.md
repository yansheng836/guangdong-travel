## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动
- **秋季（9-11月）**：天高气爽，是旅游旺季
- **夏季（6-8月）**：适合海滨活动，但需注意防晒和台风

### 推荐行程

- **一日游**：红海湾 → 品清湖 → 凤山祖庙
- **两日游**：Day1: 红海湾 → 品清湖 → 凤山祖庙；Day2: 玄武山 → 汕尾博物馆

## 交通指南

- **高铁**：汕尾站（厦深高铁）
- **自驾**：沈海高速汕尾出口
