## 地图信息

中山市位于广东省中南部，珠江三角洲腹地，东临珠江口，南接珠海市，西临江门市，北连广州市。中山市是著名的侨乡，也是伟大的民主革命先驱孙中山先生的故乡。

## 旅游建议

- 建议游览时间：2-3天
- 最佳旅游季节：春季（3-5月）和秋季（9-11月）
- 交通方式：中山市交通便利，可乘坐高铁、汽车或飞机到达
- 住宿建议：中山市区酒店众多，可根据预算选择合适的住宿
