## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动
- **秋季（9-11月）**：天高气爽，是旅游旺季
- **夏季（6-8月）**：适合海滨活动，但需注意防晒和台风

### 推荐行程

- **一日游**：汕头小公园 → 汕头博物馆 → 礐石风景区
- **两日游**：Day1: 汕头小公园 → 汕头博物馆 → 礐石风景区；Day2: 南澳岛 → 青澳湾

### 美食推荐

- 潮汕牛肉火锅
- 潮汕肠粉
- 潮汕粿条
- 南澳海鲜
- 潮汕工夫茶

## 交通指南

- **飞机**：揭阳潮汕机场，距汕头市区约35公里
- **高铁**：汕头站、潮汕站
- **自驾**：沈海高速、汕昆高速等可达
