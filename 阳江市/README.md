## 城市概况

阳江市位于广东省西南部，是广东省重要的滨海旅游城市。阳江市拥有丰富的海洋资源和独特的喀斯特地貌，其中海陵岛是广东省著名的滨海旅游度假区，春湾-凌霄岩则以其壮观的溶洞景观吸引着众多游客。

## 旅游特色

- **滨海风光**：海陵岛、十里银滩等优质海滩
- **溶洞奇观**：春湾-凌霄岩等喀斯特地貌景观
- **渔港文化**：闸坡渔港等传统渔港风情
- **人文历史**：阳江博物馆等文化景点
