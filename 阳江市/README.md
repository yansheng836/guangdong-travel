## 旅游特色

- **滨海风光**：海陵岛、大角湾、十里银滩、北洛秘境、敏捷黄金海岸等优质海滩
- **溶洞奇观**：春湾-凌霄岩、玉溪三洞等喀斯特地貌景观
- **渔港文化**：闸坡渔港、大澳渔村等传统渔港风情
- **红色旅游**：潭簕红色展馆、边海红色展馆、横山红色展馆等红色教育基地
- **工业旅游**：阳江十八子、张小泉工业旅游景区等特色工业旅游
- **温泉养生**：咸水矿温泉、阳江温泉、阳春温泉等温泉度假胜地
