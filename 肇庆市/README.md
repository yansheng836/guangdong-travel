## 旅游建议

- **一日游路线**：七星岩 → 肇庆古城墙 → 包公祠 → 肇庆博物馆
- **两日游路线**：Day1: 七星岩 → 鼎湖山；Day2: 肇庆古城墙 → 包公祠 → 肇庆博物馆
- **特色体验**：龙母诞期间（农历五月初八）前往德庆龙母庙体验民俗活动

## 交通指南

- **高铁**：肇庆东站、肇庆站
- **高速公路**：广肇高速、江肇高速
- **市内交通**：公交、出租车、共享单车
