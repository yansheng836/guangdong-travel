## 景点分布

### 端州区
- [七星岩](肇庆市-端州区-七星岩.md) - 5A级风景区，以喀斯特地貌和湖光山色闻名
- [肇庆古城墙](肇庆市-端州区-肇庆古城墙.md) - 广东省保存最完整的宋代城墙
- [包公祠](肇庆市-端州区-包公祠.md) - 纪念北宋名臣包拯的历史建筑
- [肇庆博物馆](肇庆市-端州区-肇庆博物馆.md) - 肇庆市最大的综合性博物馆

### 鼎湖区
- [鼎湖山](肇庆市-鼎湖区-鼎湖山.md) - 中国第一个国家级自然保护区

### 德庆县
- [德庆龙母庙](肇庆市-德庆县-德庆龙母庙.md) - 珠江三角洲地区最著名的龙母庙

### 封开县
- [封开国家地质公园](肇庆市-封开县-封开国家地质公园.md) - 广东省第一个国家地质公园，拥有"天下第一石"大斑石

### 鼎湖区（补充）
- [砚洲岛](肇庆市-鼎湖区-砚洲岛.md) - 西江最大江心岛，因包拯"掷砚成洲"传说而得名

### 广宁县
- [广宁竹海](肇庆市-广宁县-广宁竹海.md) - 广东省最大的竹海景区，中国"竹子之乡"

### 四会市
- [四会奇石河](肇庆市-四会市-四会奇石河.md) - 以奇石、瀑布为主题的4A级景区

## 旅游建议

1. **一日游路线**：七星岩 → 肇庆古城墙 → 包公祠 → 肇庆博物馆
2. **两日游路线**：
   - 第一天：七星岩 → 鼎湖山
   - 第二天：肇庆古城墙 → 包公祠 → 肇庆博物馆
3. **特色体验**：龙母诞期间（农历五月初八）前往德庆龙母庙体验民俗活动

## 交通信息

- **高铁**：肇庆东站、肇庆站
- **高速公路**：广肇高速、江肇高速
- **市内交通**：公交、出租车、共享单车
