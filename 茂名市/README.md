## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动和海滨游玩
- **秋季（9-11月）**：天高气爽，是旅游旺季

- **夏季（6-8月）**：适合海滨活动，但需注意防晒和台风

### 推荐行程

- **一日游A**：中国第一滩 → 晏镜岭 → 浪漫海岸
- **一日游B**：茂名露天矿生态公园 → 茂名森林公园

- **两日游**：Day1: 茂名露天矿生态公园 → 中国第一滩 → 晏镜岭；Day2: 浪漫海岸 → 放鸡岛
- **三日游**：Day1: 茂名露天矿生态公园 → 茂名森林公园；Day2: 高州玉湖风景区 → 高州冼太庙；Day3: 信宜天马山 → 窦州古城

## 交通指南

- **高铁**：茂名站、茂名西站
- **自驾**：沈海高速、包茂高速等可达
