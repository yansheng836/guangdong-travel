## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动
- **秋季（9-11月）**：天高气爽，是旅游旺季
- **夏季（6-8月）**：可登凤凰山避暑，但需注意台风

### 推荐行程

- **一日游**：潮州古城 → 牌坊街 → 开元寺 → 广济桥
- **两日游**：Day1: 潮州古城 → 牌坊街 → 开元寺 → 广济桥；Day2: 韩文公祠 → 潮州博物馆 → 凤凰山

### 美食推荐

- 潮州牛肉火锅
- 潮州肠粉
- 潮州粿条
- 潮州工夫茶
- 潮州卤味

## 交通指南

- **飞机**：揭阳潮汕机场，距潮州市区约30公里
- **高铁**：潮汕站、潮州站
- **自驾**：沈海高速、汕昆高速等可达
