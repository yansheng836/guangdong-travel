## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动和海滨游玩
- **秋季（9-11月）**：天高气爽，是旅游旺季
- **冬季（12-2月）**：气候温暖，适合避寒旅游

### 推荐行程

- **一日游**：湖光岩 → 湛江观海长廊
- **两日游**：Day1: 湖光岩 → 硇洲岛；Day2: 湛江观海长廊 → 湛江博物馆

### 美食推荐

- 湛江海鲜（生蚝、对虾、螃蟹）
- 湛江白切鸡
- 湛江牛腩粉

## 交通指南

- **高铁**：湛江西站、湛江站
- **自驾**：沈海高速、汕湛高速等可达
- **飞机**：湛江吴川机场
