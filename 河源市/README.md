# 河源市旅游景点

河源市位于广东省东北部，东江中上游，是客家人的主要聚居地之一。河源历史悠久，文化底蕴深厚，自然风光秀美，拥有丰富的旅游资源。这里是"中华恐龙之乡"，也是华南地区重要的生态屏障和水源涵养地。
