# guangdong-travel

搜集、整理广东省的旅游景点，包括：A级景区、名胜古迹、博物馆、艺术馆、图书馆、名山、著名公园等。

每个景点需要包含：景点介绍、特点、位置、通行，最好是有图片。

每个景点整理为单个文件，命名规则：市级目录/市-区-景点名，比如：广州市/广州市-海珠区-广州塔。
