## 旅游建议

### 最佳旅游时间

- **春季（3-5月）**：气候宜人，适合户外活动
- **秋季（9-11月）**：天高气爽，是旅游旺季
- **夏季（6-8月）**：海滨游玩、南昆山避暑的好时节

### 推荐行程

- **一日游**：惠州西湖 → 红花湖骑行 → 丰渚园
- **两日游**：Day1：罗浮山 → 五矿·哈施塔特；Day2：巽寮湾 → 双月湾 → 海龟湾
- **三日游**：Day1：惠州西湖 → 东坡祠 → 水东街；Day2：罗浮山 → 五矿·哈施塔特；Day3：巽寮湾 → 海滨温泉 → 双月湾

### 美食推荐

- 客家窑鸡
- 梅菜扣肉
- 惠州酿豆腐
- 东江盐焗鸡
- 惠东海鲜

## 交通指南

- **飞机**：惠州平潭机场，有飞往全国主要城市的航线
- **高铁**：惠州站、惠州南站、惠东站
- **自驾**：广惠高速、惠河高速、深汕高速等可达
